JS学习记录（持续更新）

JavaScript
一、概览
1.JavaScript主要包含ECMAScript,DOM和BOM三部分。
2.书写JS：
区分大小写；
标识符规范；
忽略程序种标识之间的空格；
忽略换行符；
以分号结尾；
单行注释与多行注释；
采用unicode编写。

二、数据类型
1.Number,Boolean,String,Object,Undefined,Null,(ES6新增Symbol)
2.Number
（1）JS不区分浮点型与整形
（2）具体采用IEEE 754标准定义的64为位浮点数表示数字。
（3）整数用整形存储，小数用浮点型存储。
（4）NaN表示非数字。
2.1算术运算
（1）基础算术运算（+-/%）
（2）封装在Math中的复杂运算：
Math.pow(a,b):a的b次方
Math.round(n):四舍五入
Math.ceil(n):向上取整
Math.floor(n):向下取整
Math.abs(n):求绝对值
Math.max(x,y,z):返回最大值
Math.min(x,y,z):返回最小值
2.2.Date()构造函数创建日期
3.String
3.1字符串直接量
（1）由单引号或双引号括起来的字符序列。
（2）引号内可直接包括引号不用转义。
3.2转义字符
（1）反斜线\加一个字符。
3.3匹配字符
（1）两条斜线之间的文本构成了一个正则表达式直接量。
（2）第二条斜线后可以跟修饰符。
（3）正则表达式，JS采用Perl的正则语法。
/^HTML/ 匹配以HTML开头的字符串
/[1-9][0-9]/ 匹配一个非零数字，后面是任意个数字
/\bjavascript\b/i 匹配单词javascript，忽略大小写
（4）RegExp和String对象均定义了利用正则表达式进行模式匹配和查找与替换的函数。
4.其他数据类型
4.1布尔类型
只有两个值true和false
4.2对象
（1）对象是一组数据和功能的集合。
（2）对象的创建可通过构造函数进行。
var obj=new Object();var obj={};
（3）每一个对象都可以自定义属性和方法。
（4）可用instanceof来判断对象是否存在。
4.3Null和Undefined
（1）Undefined只有一个值，即undefined，变量未初始化时，默认为undefined。
（2）Null也只有一个值，即null，表示一个空对象指针。
5数据类型检测：typeof表达式
var a,b=null,c=new Object();var d=function(){};
typeof ‘123’//string
typeof true//boolean
typeof a//undefined
typeof b//Object
typeof c//Object
typeof d//Function

三、类型转换
10+’objects’//‘10objects’，数字10转换成字符串；
‘7’*’4’//28，两个字符串均转换成数字；
var n=1-‘x’//NaN字符串’x’无法转换成数字；
n+’objects’//NaN objects;NaN转换成字符串’NaN’；
1相等性：以下结果均为true
null == undefined;//这两值被认为相等
“0” ==0;//比较前字符串转数字
0 == false;//比较前布尔值转数字
“0” == false;//比较前布尔值和字符串都转数字
2.显示转换
（1）使用Boolean(),Number(),String(),Object()函数。当不通过new运算符调用这些函数时，他们会作为类型转换函数并按照规则做类型转换。
（2）eg：
Number(“3”)//3；parselnt()和parseFloat()函数也可用于数字转换；
String(false)//“false”；或使用false.toString()；
3.转换为原始值
（1）JS的数据类型可以分为两种，原始类型和引用类型。原始类型为基本类型，引用类型为对象。
（2）所有的对象继承了两个转换方法，第一个是toString()，返回这个对象的字符串。
({x:1,y:2}).toString()//“[object Object]”
（3）另一个转换对象的函数是valueOf()。如果存在任意原始值，它就默认将对象转换为表示他的原始值。对象是复合值，而且大多数对象无法真正表示为一个原始值，因此默认的valueOf()方法简单地返回对象本身，而不是返回一个原始值。

四、变量声明
1.1var声明
（1）在JS中一般使用var关键词声明变量：
var carName；
（2）变量声明后，其值默认为空。
（3）在JS中，默认在最开始对所有变量进行声明，不管其在程序中的实际位置，这叫做变量提升。
（4）var声明的变量不存在块级作用域。
1.2直接声明全局变量
carName=value//严格模式下报错
2命名规则
（1）以字母、$、_开头。
（2）变量中其他位置可以有数字。
（3）大小写敏感。
3.变量值如何放入HTML标签

五、作用域
1.变量与函数的可访问范围，分为全局作用域和局部作用域。
2.全局作用域
（1）最外层定义的函数或变量。
（2）局部作用域中的变量不适用var关键词声明。
（3）所有window对象属性拥有全局作用域。
3.局部有作用域
代码片段或函数内部。
4.作用域链
[[Scope]]内部属性包含了函数被创建的作用域中对象的集合，这个集合称为函数的作用域链，决定了哪些数据能被函数访问。函数执行过程中，每遇到一个变量，都会经历一次标识符解析过程以决定从何处获取和存储数据。解析过程优先从作用域链中的局部变量进行选取。

六、原始表达式
1.原始表达式包含变量名和运算符。
2.原始表达式是表达式的最小单位。
3.直接量。数字直接量(1.2)字符串直接量(‘aa’)正则表达式直接量(/pattern/)
4.关键字构成的原始表达式。true,false,null,this

七、函数定义表达式
1.函数定义：赋值后才可调用。
var square =function(){}
2.函数声明：定义前就可调用。
Function functionName(){}
3.函数调用与对象创建
（1）对象创建表达式创建一个对象并调用一个(构造)函数初始化新对象的属性。
（2）对象创建表达式和函数调用表达式类似，区别是对象创建表达式之前多了一个new关键字。
new Object();
（3）如果没有参数，可以省略括号。
new Date;

八、运算符
1.由标点符号和关键字来表示。
2.常见运算符：
++,–,-,+,~,|,delete,typeof,void

九、表达式
1.算术表达式
一元加法：把操作数转化为数字
一元减法：先转化为数字再取反
2.关系表达式
=：得到或者赋值
==：检测是否相等，可以允许进行类型转换，求反：!=
===：恒等，检测两个操作数是否严格相等，求反!==
in：左操作数是一个字符串或者可转化为字符串，右操作数是一个对象。如果右操作数的对象拥有一个名为左操作数的属性名，那么表达式返回true。
instanceof：左操作数是一个对象，右操作数标识对象的类。如果左侧的对象是右侧类的实例，则返回true。
3.逻辑表达式
&&，||，！，存在短路规则。
4.赋值表达式
多个赋值运算符从右到左计算。
